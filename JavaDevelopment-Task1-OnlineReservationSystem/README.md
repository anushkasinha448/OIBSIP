# Online Reservation System

## Oasis Infobyte Internship

### Task 1 - Online Reservation System

## Project Description

This project is a Java Swing based Online Reservation System developed as part of the Oasis Infobyte Java Development Internship. It allows users to securely log in, search trains, reserve tickets, generate unique PNR numbers, view booking history, and cancel reservations using JDBC with an Oracle database.

## Features

- User Login
- User Registration
- Search Train by Train Number
- Ticket Reservation
- Automatic PNR Generation
- Booking Confirmation
- Booking History
- Ticket Cancellation using PNR
- Input Validation

## Technologies Used

- Java
- Java Swing
- JDBC
- Oracle Database
- Eclipse IDE

## Project Structure

- src/
- lib/
- screenshots/
- database.sql

## How to Run

1. Import the project into Eclipse.
2. Add the Oracle JDBC driver (ojdbc11.jar).
3. Create the required database tables using `database.sql`.
4. Update the database credentials in `DBConnection.java`.
5. Run `Main.java`.
