## 📂 Projects

### 📌 Task 1 – Online Reservation System

A Java Swing-based Online Reservation System that enables users to log in, reserve train tickets, generate unique PNR numbers, view booking history, and cancel reservations. The project uses **JDBC** with an **Oracle Database** for data storage and retrieval.

**Technologies Used:**
- Java
- Java Swing
- JDBC
- Oracle Database
- Eclipse IDE

### 📌 Task 3 – ATM Interface

A Java Swing-based ATM Interface that simulates ATM operations such as secure login, balance enquiry, cash withdrawal, deposit, fund transfer, and transaction history. The application uses **JDBC** and **Oracle Database** for persistent data management.

**Technologies Used:**
- Java
- Java Swing
- JDBC
- Oracle Database
- Eclipse IDE

## 🚀 How to Run

1. Clone this repository.
2. Open the desired project in Eclipse.
3. Add the Oracle JDBC Driver (`ojdbc11.jar`) to the project's build path.
4. Create the required database tables using the provided `database.sql` file.
5. Update the database credentials in `DBConnection.java`.
6. Run the `Main.java` file to launch the application.
